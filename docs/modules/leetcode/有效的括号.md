## 题目

给定一个只包括 `'('`，`')'`，`'{'`，`'}'`，`'['`，`']'` 的字符串 `s` ，判断字符串是否有效。

有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。
3. 每个右括号都有一个对应的相同类型的左括号。

**示例 1：**

```
输入：s = "()"
输出：true
```

**示例 2：**

```
输入：s = "()[]{}"
输出：true
```

**示例 3：**

```
输入：s = "(]"
输出：false
```



**提示：**

- `1 <= s.length <= 104`
- `s` 仅由括号 `'()[]{}'` 组成

## 代码实现

### go实现

~~~go
func isValid(s string) bool {
	n := len(s)
	if n % 2 == 1 {
		return false
	}
	p := map[byte]byte {
		')': '(',
		']': '[',
		'}': '{',
	}
	stack := []byte{}
	for i := 0; i < n; i ++ {
		// 存在, 说明这个字符是要移除 堆栈 里的数据
		if p[s[i]] > 0 {
			if len(stack) == 0 || stack[len(stack) - 1] != p[s[i]] {
				return false
			}
			stack = stack[:len(stack) - 1]
		} else {
			stack = append(stack, s[i])
		}
	}

	return len(stack) == 0
}
~~~
